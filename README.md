# دفتر پرسش‌ها

یک سایت استاتیک و راست‌به‌چپ برای نمایش ۵۷۴ پرسش و راهنما.

## اجرا

برای اجرای محلی:

```bash
python3 -m http.server 4173
```

سپس این آدرس را باز کنید:

```text
http://localhost:4173
```

## انتشار

این پروژه بدون build step روی GitHub Pages اجرا می‌شود. کافی است فایل‌ها از شاخه `main` منتشر شوند.
